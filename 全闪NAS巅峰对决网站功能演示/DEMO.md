## 📱 网站访问指南

### 打开方式
直接双击 `index.html` 文件即可在浏览器中打开网站！

### 本地服务器部署（推荐）
```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 然后在浏览器中访问
# http://localhost:8000
```
