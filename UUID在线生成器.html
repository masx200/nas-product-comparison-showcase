UUID在线生成器-工具简介

UUID在线生成器，GUID在线生成器工具(GUID是微软对UUID的一种称呼)。支持生成Version 1和Version4算法的UUID。一般使用Version4 UUID。

UUID在线生成器-使用说明

UUID在线生成器，支持生成多个版本的 UUID，支持批量生成 UUID,并支持下载到文件。

  1. UUID版本：选择生成的 UUID 版本。本工具支持生成 Version 1,Version 3,Version 4,Version 5,Version 6,Version 7 版本的 UUID 。
  2. 生成数量：填写 UUID 生成的数量，数量的范围为 [1,1000000] 。
  3. 结果格式：选择 UUID 的结果格式化方式，支持 string,hex,binary,base64 格式。
  4. UUID 英文百科介绍， 中文百科介绍。
  5. UUID是国际标准化组织（ISO）提出的一个概念。UUID是一个128比特的数值，这个数值可以通过一定的算法计算出来。 UUID用来识别属性类型，在所有空间和时间上被视为唯一的标识。一般来说，可以保证这个值是真正唯一的， 即任何地方产生的任意一个UUID都不会有相同的值。
  6. UUID Version 1：基于时间的UUID，通过计算当前时间戳、随机数和机器MAC地址得到。
  7. UUID Version 2：基于时间和 MAC 地址生成 UUID，DCE 安全版本，RFC 9562 保留版本，未提供具体的实现方式。
  8. UUID Version 3：基于命名空间和名称生成 UUID，使用 MD5 哈希算法。对于一个给定的命名空间和名称，生成的 UUID 为定值。
  9. UUID Version 4：随机UUID，根据随机数，或者伪随机数生成UUID。
  10. UUID Version 5：基于命名空间和名称生成 UUID，使用 SHA1 哈希算法。对于一个给定的命名空间和名称，生成的 UUID 为定值。
  11. UUID Version 6：基于时间的UUID，通过计算当前时间戳、随机数和机器MAC地址得到。
  12. UUID Version 7：基于时间戳和随机数生成，适用于高负载的数据库和分布式系统。
  13. UUID Version 8：用户自定义 UUID 生成。
